# AlgoVision

An interactive **Sorting Algorithm Visualizer** built with **React + Vite** that helps users understand how different sorting algorithms work through real-time animations, performance statistics, and algorithm-specific visualizations.

Designed with an intuitive UI and smooth animations, AlgoVision transforms abstract sorting concepts into engaging visual experiences for students, educators, and competitive programmers.

---

## 📖 Overview

AlgoVision is a web-based visualization tool that demonstrates the internal working of popular sorting algorithms. Users can generate random arrays, customize inputs, control animation speed, pause/resume execution, and observe algorithm performance in real time.

Unlike traditional visualizers, AlgoVision also includes a dedicated **Merge Sort Tree Visualization**, allowing users to understand the recursive divide-and-conquer process step by step.

## ⚙️ How It Works

1. Select a sorting algorithm.
2. Generate a random array or enter a custom one.
3. Adjust the array size and animation speed.
4. Start the visualization.
5. Watch comparisons, swaps, and recursive operations update live.
6. View execution statistics after completion.
